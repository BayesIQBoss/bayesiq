# ADR-0004: Sonos control requires approval and guardrails
Status: Accepted
Date: 2026-02-28

## Decision (Guardrails)
- allowed rooms: Living Room, Dining Room, Kitchen
- max volume: 40/100
- quiet hours: disabled
- approval required for any execute action

## Consequences
- Prevents accidental or disruptive actions
- Slight interaction overhead, acceptable for safety
