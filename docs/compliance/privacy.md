# Privacy (v0.1)

## Data stored
- event logs and tool run summaries (redacted)
- calendar event metadata needed for agenda formatting
- PR metadata (repo, branch, PR URL)

## Data not stored
- raw OAuth tokens in logs
- full calendar descriptions unless required (default: omit)
- any audio/voice data (out of scope)

## Access
- local machine + encrypted backups
- admin profile controls approvals
